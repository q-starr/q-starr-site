Starr AI downloads an open-source language model (1–2 GB) to your iPhone during first launch. After that, all inference runs on Apple's Neural Engine — the same chip that powers Face ID and photo processing. No server needed.
Starr AI works on any iPhone running iOS 17+. For the best experience with the default Phi-4 Mini model, iPhone 15 Pro or newer is recommended (8GB RAM). The SmolLM2 model runs well on older devices with 6GB RAM.
ChatGPT uses much larger models running on cloud servers, so it's stronger on complex reasoning and long-form analysis. Starr AI uses compact on-device models — it's great for everyday questions, translation, and brainstorming, and it works when ChatGPT literally can't (no internet).
